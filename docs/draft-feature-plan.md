# 📝 입찰공고 제안서 초안 작성 기능 기획서

## 1. 개요

입찰공고 리스트에서 **원클릭으로 AI가 제안서 초안을 생성**해주는 기능입니다.
공고의 제목·기관·예산 등 기본 정보와 함께, 사용자가 **RFP(제안요청서)나 평가기준을 직접 입력**하면 더 정확한 맞춤형 초안을 생성합니다.

> ⚠️ AI가 생성하는 초안은 **참고용**이며, 실제 제출 전 반드시 전문가 검토가 필요합니다.

---

## 2. 사용자 흐름 (User Flow)

```
공고 리스트 → [✍️ 초안 작성] 클릭
→ [1단계] 양식 선택 + RFP 입력 → [🤖 AI 초안 생성하기] 클릭
→ [2단계] 로딩(10~30초) → 초안 결과 표시
→ [📋 복사] / [⚙️ 설정 변경] / [🔄 다시 생성]
```

### 단계별 상세

| 단계 | 화면 | 설명 |
| :--- | :--- | :--- |
| 1 | 공고 카드/모달 | `✍️ 초안 작성` 버튼 클릭 |
| 2 | 설정 모달 | 제안서 양식 선택 (3종) + RFP 텍스트 입력 (선택) |
| 3 | 설정 모달 | `🤖 AI 초안 생성하기` 클릭 |
| 4 | 결과 모달 | 로딩 → 마크다운 초안 렌더링 |
| 5 | 결과 모달 | 복사 / 설정 변경 / 다시 생성 / 닫기 |

---

## 3. 제안서 양식 (3종)

| 양식 | 아이콘 | 설명 | 섹션 수 |
| :--- | :---: | :--- | :---: |
| **기술제안서** | 🔧 | 기술 역량·방법론·아키텍처 중심 | 6개 |
| **사업계획서** | 📊 | 사업 타당성·ROI·전략 중심 | 6개 |
| **간이제안서** | 📝 | 소규모 사업용 간결한 양식 | 4개 |

---

## 4. RFP 입력 기능

사용자가 공고 내 **제안요청서(RFP), 과업내용서, 평가기준** 등을 텍스트로 붙여넣으면:
- AI가 평가항목별 배점에 맞춰 비중 조절
- 요구사항에 맞는 구체적인 수행방안 생성
- 과업 범위에 맞춘 일정·인력 계획 제시

**미입력 시**에는 공고 제목에서 핵심 키워드를 추출하여 사업 범위를 추정합니다.

---

## 5. 기술 구현

```
[프론트엔드] page.tsx
  └─ ✍️ 초안 작성 버튼 → 설정 모달 (1단계)
  └─ 양식 선택 + RFP 입력 → POST /api/generate-draft
  └─ 결과 모달 (2단계) 에서 렌더링

[백엔드] /api/generate-draft/route.ts
  └─ 요청 검증 → 양식별 프롬프트 조합 → Gemini API 호출 → 마크다운 응답

[AI] Gemini 2.5 Pro
  └─ 공고 정보 + RFP 컨텍스트 + 양식 구조 → 맞춤형 제안서 초안
```

---

## 6. 향후 고도화 계획

### Phase 2: 회사 프로필 연동
- 사전 등록된 회사 정보(실적, 역량, 인력)를 프롬프트에 반영
- 더 개인화된 맞춤형 초안 생성

### Phase 3: 첨부 파일 분석
- 공고 첨부 제안요청서(RFP) PDF를 업로드하여 분석
- RFP의 평가 기준에 맞춰 초안 최적화

### Phase 4: HWPX 문서 출력
- 생성된 초안을 HWPX(한글) 파일로 변환
- 제안서 템플릿에 자동 삽입하여 다운로드

---

## 7. 주의사항

> [!CAUTION]
> - AI 초안은 **참고 자료**입니다
> - 실제 RFP를 직접 입력하면 더 정확하지만, AI 추론의 한계가 있습니다
> - **반드시 전문가 검토 후** 실제 제안서에 활용하세요
